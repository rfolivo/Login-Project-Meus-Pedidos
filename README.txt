Projeto LOGIN

Esse projeto foi desenvolvido através da ferramenta Selenium Builder.

Passos para executar o teste:

1 - Instalar o plugin - Selenium Builder no Firefox versão 39
2 - Abrir a ferramenta e selecionar a opção File > Open a script or Suite.
3 - Importar o arquivo Suite-Login.json
4 - Após incluir o projeto, executar na opção: Run > Run suite locally.
5 - Todos os scritps serão executados na ordem correta.
6 - Os detalhes dos passos de cada script está no documento Roteiro de testes.xls
